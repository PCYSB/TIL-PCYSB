# DNS(Domain Name System)

DNS는 도메인 이름과 IP 주소의 대응표를 관리하는 체계이다. 도메인은 IP 주소를 외우기 힘들기에 나왔다.

# 과정(간략히)

1. 어떠한 주소를 검색
2. 내 PC 메모리를 뒤짐 DNS Cache를 뒤짐
3. 그걸 검색했었는지 확인
4. host file을 뒤져봄
5. DNS한테 물어봄
